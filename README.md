# ContaBancaria

Um repositório contando um pequeno projeto envolvendo classes Python.

Nele temos uma classe Conta, que é pai de Conta Poupança e Conta Corrente, cada qual com suas peculiaridaes

Na poupança podemos verificar o quanto o dinheiro rende ao ano, com uma taxa de 47,5%.
Na corrente, podemos ter um saldo especial (limite) e realizar e pagar empréstimos.

Ambas possuem opção de transferir o saldo entre si, desde que se acesse cada uma primeiro para obter seu ID, necessário para a transferência.
